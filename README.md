# stories

Un catálogo de relatos disponibles para leer y descargar de forma gratuita.

📖 Sobre el proyecto

Este sitio reúne cuentos escritos por mi.
Todos los textos están disponibles sin costo, con la idea de que las historias circulen y lleguen a quien quiera leerlas.

📂 Contenido
Relatos en formato EPUB y PDF
Catálogo organizado
Descarga libre
🤍 Apoyo

Si te gustó algún relato y querés apoyar mi escritura, dentro del sitio hay opciones para hacerlo.
